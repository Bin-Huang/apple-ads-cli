# apple-search-ads-cli

A Apple Search Ads CLI designed for AI agents. Wraps the official Apple Search Ads API with simple, agent-friendly commands.

**Works with:** OpenClaw, Claude Code, Cursor, Codex, and any agent that can run shell commands.

> **Status: Under Development** — This project is not yet functional. Star/watch for updates.

## API Reference

- Official docs: https://developer.apple.com/documentation/apple_search_ads
